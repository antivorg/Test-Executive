
# Test-Executive
A fully configurable multithreaded GUI for test software running/automation/regression, designed to interact with existing code bases with little to no modification.

## Goals:
	- Generic file handle(s) interface as an input
	- Being able to diplays said file handle(s) output as quickly as possible
	- Provide a generic interface for interpreting file handle(s) contents
	- Provide a generic interface for updating status
